This folder contains the Jupyter notebooks used for data cleaning, analysis, and feature engineering.
Data Source: National Bureau of Statistics (Nigeria). Monthly Selected Fod Price Watch reports and related datasets. Data can be obtained from the NBS website.
- Select Food Price Watch: https://nigerianstat.gov.ng/elibrary?queries=food%20price%20watch
- PMS Price Watch: https://nigerianstat.gov.ng/elibrary?queries=Premium%20Motor%20Spirit
- Exchange Rate Watch: https://www.cbn.gov.ng/rates/ExchRateByCurrency.html
